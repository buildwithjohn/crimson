# CrimsonWings — Official Website

**Delivering Blood. Saving Lives. On Time, Every Time.**

Nigeria's most advanced digitized blood bank and integrated medical logistics platform.

## Tech Stack
- **Framework**: Next.js 15 (App Router, Static Export)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Custom CSS Design System
- **Animations**: CSS Intersection Observer, Custom keyframes
- **Fonts**: Playfair Display (display) + DM Sans (body) + DM Mono (mono)
- **Icons**: Lucide React

## Design Pillars
1. **Clinical Authority** — NAT screening, cold chain, compliance
2. **Operational Scale** — 20K+ units/month, hub-and-spoke
3. **Technological Superiority** — Proprietary OS, triphasic drone logistics

## Sections
- Hero — animated with scan line, stats grid, ticker
- About — mission and founding numbers
- The Problem — Nigeria blood crisis
- Solution — 3 pillars
- Technology — NAT, OS platform, drone delivery
- How It Works — 6-phase donor-to-delivery flow
- Services — 6 service lines
- Impact — animated counters
- Partnerships — stakeholders + investor CTA
- FAQ — accordion
- Contact — form + details
- Footer
